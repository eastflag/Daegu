네이버 사용 방법

1) 인터넷 아이콘을 찾아서 클릭

2) 주소창을 클릭하고 m.naver.com을 입력

3) 로딩 완료

Back

사진 다운로드 방법

1) 다운로드 받을 사진을 길게 누른다.

2) 팝업창이 뜨면 저장을 클릭한다.

Back

글 복사하는 방법

1) 복사할려는 문장 시작 지점을 롱클릭

2) 핀 아이콘이 두개가 생기면 핀아이콘을 잡아서 움직여서 영역을 선택한 후 복사를 클릭

Back

친구 추가 방법

1) 카카오톡을 실행.

2) 돋보기 아이콘이 있는 검색 탭을 클릭 후 아이디 검색을 클릭

3) 검색할 친구 아이디를 입력후 추가 클릭

4) 검색결과가 맞으면 해당 아이디를 클릭

5) 추가 버튼을 클릭

Back

메시지 전송 방법

1) 친구탭을 클릭 후 대화하고자하는 친구를 클릭

2) 1:1 채팅을 클릭

3) 대화 메시지 전송

Back

사진 보내는 방법

1) 대화입력창 옆의 + 버튼 클릭 후 사진 아이콘 클릭

2) 전송할려는 사진을 여러개 선택 후 -> 클릭

3) 전송 버튼 클릭

4) 전송 확인

Back

WiFi 켜는 방법

1) 상단 상태바를 클릭후 아래로 드래그

2) Wi-Fi 버튼 클릭

Back

메시지 보내는 방법

1) 메시지 아이콘 클릭

2) 새 메시지 작성 아이콘 클릭

3) 받는 사람과 메시지 입력

4) 사진을 전송할 경우 첨부 아이콘 클릭후 사진 선택

5) 전송 버튼 클릭
